# TamStore 🛒

[View TamStore Website](https://nimble-cranachan-1e97e3.netlify.app/)

## 📖 Overview
The **TamStore** is a modern e‑commerce web application built with **React**. It integrates **API services**, global state management using  **Zustand** , **Clean Architecture**, and provides a smooth user experience with **Framer Motion animations**. The app includes authentication, protected routes, wishlist management, and full CRUD functionality in the Profile page.

---

## 🚀 Features
- **Authentication**: Secure login/register with protected routes.  
- **Home Page**: Landing page showcasing products and navigation.  
- **Products Page**: Displays product listings fetched from API.  
- **Cart Page**: Add/remove items, manage quantities, and checkout flow.  
- **Wishlist**: Context‑based wishlist provider for saving favorite items.  
- **Profile Page**: Full **CRUD operations** (Create, Read, Update, Delete) for user data.  
- **Error Boundary**: Graceful error handling across the app.  
- **Theme Provider**: Dark/Light mode toggle.  
- **Framer Motion**: Page transitions and animations for smooth navigation.  
- **Toastify**: Notifications for user actions (add to cart, login success, etc.).  

---

## 🛠️ Tech Stack
- **React** (with React Router v6)  
- **Zustand** (state management)  
- **Framer Motion** (animations)  
- **React Toastify** (notifications)  
- **Context API** (Wishlist, Theme)  
- **Error Boundaries** (robust error handling)  
- **API Integration** (products, auth, profile CRUD)  
- **Clean Architecture**
